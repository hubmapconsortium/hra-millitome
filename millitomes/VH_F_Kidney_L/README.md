# Millitome files for Visual Human Female Left Kidney

This directory contains compressed files. Each ZIP file will expand into a folder containing all necessary files to 3D-print millitomes for the a specific sample block size. Also included are matching CSV files for sample block registration.

<p>For example <em>VH_F_Kidney_L_10_Millitomes.zip</em> contains all files needed to print 10mm block size millitomes.</p>

<p>The content of this specific ZIP file would be:</p>

VH_F_Kidney_L_10_Small_Top.stl<br>
VH_F_Kidney_L_10_Large_Bottom.stl<br>
VH_F_Kidney_L_10_Large_Top.stl<br>
VH_F_Kidney_L_10_linear.csv<br>
VH_F_Kidney_L_10_matrix.csv<br>
VH_F_Kidney_L_10_Medium_Bottom.stl<br>
VH_F_Kidney_L_10_Medium_Top.stl<br>
VH_F_Kidney_L_10_Small_Bottom.stl<br>
